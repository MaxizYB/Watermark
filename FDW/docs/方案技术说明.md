# FDW 成熟方案技术说明

> 版本：v3 最终版 | 更新：2026-05-17
> 模型：Stable Diffusion 2.1-base | 采样器：DDIM 50步 | FPR = 10⁻⁶

---

## 一、方案概述

FDW（Frequency-Domain Watermarking）是一种面向扩散模型的无训练水印方案，在 Gaussian Shading 基础上引入频域模板注入和几何盲校正，在保持图像质量无损的前提下实现了对旋转、缩放等几何攻击的鲁棒检测。

### 核心参数

| 参数 | 值 | 含义 |
|------|-----|------|
| `hw_factor` | 4 | 空间冗余倍数（每 bit 重复 16 次） |
| `payload_bits` | 512 | 有效载荷位数 |
| `use_ecc` | True | 启用 2× 重复码（实际 1024 bit） |
| `lambda_freq` | 0.02 | 初始噪声频域增强强度 |
| `alpha_max` | 0.0 | 去噪过程中 FDSC 注入强度（关闭） |
| `template_gamma` | 4.0 | X 模板浅层注入强度 |
| `template_t_start` | 0.3 | 注入时刻 t*（ShallowDiffuse 模式） |
| `fdsc_t_start/end` | 0.2/0.6 | 频域模板检测窗口 |
| `chacha` | True | 使用 ChaCha20 流密码加密 |

## 二、系统架构

### 2.1 水印嵌入流程

```
原始水印比特 b ∈ {0,1}^512
  │
  ├─① 空间冗余扩展（hw=4, repeat 16×）+ 2× 重复码
  │     → sd ∈ {0,1}^{4×64×64} (1024 bit)
  │
  ├─② ChaCha20 流密码加密（密钥由 user_id + seed 派生）
  │     → m ∈ {0,1}^{16384}
  │
  ├─③ 截断高斯采样（Truncated Sampling）
  │     bit=0 → 从 N(-∞, 0) 采样
  │     bit=1 → 从 N(0, +∞) 采样
  │     → 初始噪声 w ∈ R^{4×64×64}
  │
  ├─④ 频域增强（fd_enhance_noise, λ=0.02）
  │     对 channel 0 的中频带 [0.1, 0.5] 叠加水印模板的 FFT 分量
  │     → 增强后的初始噪声 w*
  │
  ├─⑤ 扩散生成（DDIM 50步, guidance_scale=7.5）
  │     以 w* 为初始噪声，在 t*=0.3T 步执行：
  │
  └─⑥ X 模板浅层注入（_inject_template_shallow, γ=4.0）
        在 z_t 的 FFT 频谱中，在对称 X 形位置注入结构化脉冲
        利用 ShallowDiffuse 揭示的低秩 Jacobian 零空间特性
        → 带水印图像
```

### 2.2 水印提取流程

```
待检测图像（可能经过攻击）
  │
  ├─① 几何预校正（仅 FDW, geo_correct=True 时）
  │     ├─ Scale 检测：黑边检测 + 对称性判断 → 裁剪 + 缩放回 512×512
  │     ├─ Rotation 检测：DDIM Inversion → z_T → detect_rotation_ring()
  │     │     从 z_T 的 FFT 频谱中检测 X 模板的角度偏移
  │     └─ 4-way 暴力搜索：对称模板有 90° 歧义
  │           生成 4 个候选角度 [-det, -det±90, -det+180]
  │           每个候选：旋转校正 → DDIM Inversion → score_watermark()
  │           选择 ACC 最高的候选
  │
  ├─② VAE Encode → z_0
  │
  ├─③ DDIM Inversion（空 prompt, guidance_scale=1, 50步）
  │     → z_T ≈ w*
  │
  ├─④ 符号检测：m̂ = (z_T > 0).int()
  │
  ├─⑤ ChaCha20 解密 → sd̂
  │
  ├─⑥ 空间冗余投票（16 票多数表决）+ 2× 重复码合并
  │
  └─⑦ 计算 Bit Accuracy, TPR_detection, TPR_traceability
```

## 三、核心组件详解

### 3.1 X 模板（build_x_template）

对称 X 形频域模板，16 个脉冲点，分布在对称角度（0°/90°/180°/270° ± 偏移）上，半径覆盖 [0.2, 0.5] × w/2。

**设计要点：**
- 16 点对称 → 90° 周期性 → 可用 4-way 暴力搜索消除歧义
- γ=4.0 为甜点边界（γ=3.5 时 rotate_15 ACC=0.611 失败，γ=4.0 时 ACC=1.0）
- 注入方式：`injection = γ × mask × pattern × std(Z)` 加到 FFT 频谱上
- 注入后 clamp 到 `[-3×max|z_t|, 3×max|z_t|]` 防止数值爆炸

### 3.2 几何盲校正（detect_and_correct_geom）

完整的几何攻击校正链，无需原始图像：

1. **Scale 校正**（`detect_and_correct_scale`）
   - 黑边检测（阈值 15/255）→ 对称性判断（< 35% 不对称）
   - 裁剪内容区域 → BICUBIC 缩放回 512×512
   - 仅对对称黑边触发（区分 scale 和 crop）

2. **Rotation 校正**（`detect_and_correct_rotation`）
   - `detect_rotation_ring()`：对 z_T 做 FFT，在预设半径环 [0.15, 0.55]×w/2 上扫描角度
   - 取 FFT 幅值最大的角度作为检测结果（mod 90°）
   - 4-way 暴力搜索消除 90° 歧义（需 `score_fn`）

3. **4-way 暴力搜索**
   - 候选角度：`[-detected, -detected+90, -detected-90, -detected+180]`
   - 每个候选：图像旋转校正 → center crop 512×512 → DDIM Inversion → `score_watermark()`
   - 选 ACC 最高的候选作为最终校正结果

### 3.3 频域增强（fd_enhance_noise）

在初始噪声的中频带 [0.1, 0.5] 叠加水印模板的 FFT 分量：

```
W*_fft[..., mask] = W_fft[..., mask] + λ × T_fft[..., mask]
w* = IFFT(W*_fft)
per-channel std re-normalization → 保持高斯分布
```

λ=0.02 为最优值。λ=0.08 虽在 blur 上略优，但与 γ=4.0 叠加时产生非线性质量恶化（CLIP 从 0.27 降至 0.19）。

### 3.4 截断高斯采样（Gaussian Shading 核心）

利用逆 CDF 将二进制水印比特映射到截断高斯分布：
- bit=0 → 从 N(-∞, 0) 采样（负半轴）
- bit=1 → 从 N(0, +∞) 采样（正半轴）

经 std 归一化后，w 近似服从标准高斯分布，不影响扩散模型生成质量。

## 五、关键设计决策

### 5.1 参数选择依据

| 参数 | 选择 | 依据 |
|------|------|------|
| λ=0.02（非 0.08） | 质量扫描实验 | λ=0.08+γ=4 时 CLIP 暴跌至 0.19（非线性干扰），λ=0.02 时 CLIP=0.275 ≈ clean 0.277 |
| γ=4.0（非 8.0） | LRGTI v4 精扫 | γ=4 是甜点边界（γ=3.5→ACC=0.611 失败），且 CLIP=0.280 高于 γ=8 的 0.248 |
| α=0（非 0.015） | 质量扫描实验 | FDSC denoising 注入对检测帮助有限但损害质量 |
| t*=0.3T | t* 扫描实验 | t*=0.1 几何鲁棒性最优，t*=0.3 综合表现最均衡（CLIP 最高 0.297） |

### 5.2 4-way 暴力搜索（非穷举旋转）

对称 X 模板具有 90° 周期性，因此只需 4 个候选角度而非连续穷举。每个候选需完整 DDIM Inversion（50步），计算成本 ≈ 4× 单次检测。这在可接受范围内。

### 5.3 不使用不对称模板的原因

曾尝试不对称模板 [5,4,1,2] 消除 90° 歧义，但信号太弱导致检测失败。对称模板 + 4-way 是更可靠的方案。

### 5.4 不使用 MaXsive pred_x0 注入的原因

MaXsive 方案在 guidance_scale=1（无条件生成）下可行，但我们的 CFG（gs=7.5）导致 pred_x0 在高 t 时爆炸（ᾱ_t≈0 分母极小），注入后轨迹发散 PSNR≤8dB。

## 六、已知局限

1. **Stirmark 组合攻击**：DDIM Inversion 对插值噪声敏感，TPR=0
2. **Crop 0.6**：ACC≈0.89，低于 GS 的 0.96（GS 有更多空间冗余 hw=8）
3. **Scale 0.95**：黑边太窄（≈3%），scale 检测器未触发
4. **4-way 计算开销**：旋转检测需 4 次 DDIM Inversion（≈20s/图 on A100）

## 七、与 Gaussian Shading 的对比

| 维度 | FDW (Ours) | Gaussian Shading |
|------|-----------|-----------------|
| 水印容量 | 512 bit | 256 bit (hw=8) |
| 空间冗余 | hw=4 + 2× 重复码 | hw=8 |
| 频域增强 | ✓ (λ=0.02) | ✗ |
| 几何校正 | ✓ (X模板 + 4-way) | ✗ |
| 旋转鲁棒性 | 5°~90° ACC≥0.93 | ≈0.5（随机） |
| 缩放鲁棒性 | 0.50~0.90 ACC≥0.93 | ≈0.5（随机） |
| JPEG Q=25 | ACC≈0.97 | ACC≈0.88 |
| CLIP-Score | 0.2753 | 0.2360 |
| 检测速度 | 慢（4-way ≈20s） | 快（≈1s） |
