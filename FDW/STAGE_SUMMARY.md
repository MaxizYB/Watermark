# 阶段性总结：FDW vs Gaussian Shading

> 测试环境：SD 2.1-base，N=64，DDIM 50步，攻击库 17 项，2026-05-05
> 方案版本：ShallowDiffuse 单步注入 + 多级几何盲校正

---

## 一、方法原理

### 1.1 Gaussian Shading（GS）基线

GS 是一种无需训练的扩散模型水印方法，核心思路是**将水印信息编码进初始噪声的统计分布**，利用 DDIM Inversion 的确定性来提取。

**嵌入流程：**

```
水印比特 b ∈ {0,1}^N
  → 空间冗余扩展（repeat hw×hw 次）→ sd ∈ {0,1}^{4×64×64}
  → 流密码加密（ChaCha20 / XOR key）→ m ∈ {0,1}^{16384}
  → 截断高斯采样（Truncated Sampling）→ 初始噪声 w ∈ R^{4×64×64}
      bit=0 → 从负半轴 N(-∞, 0) 采样
      bit=1 → 从正半轴 N(0, +∞) 采样
  → 扩散生成（DDIM/DPM，以 w 为初始噪声）→ 带水印图像
```

**提取流程：**

```
带水印图像（可能经过攻击）
  → VAE Encode → z_0
  → DDIM Inversion（空 prompt，确定性）→ z_T ≈ w
  → 符号检测：m̂ = (z_T > 0).int()
  → 流密码解密 → sd̂
  → 空间冗余投票（多数表决）→ b̂
  → 与原始 b 比较，计算 Bit Accuracy
```

**关键设计：**
- **截断采样**保证 w 近似服从标准高斯分布，不影响图像质量
- **空间冗余**（hw=8 时每个 bit 重复 64 次）通过多数投票抵抗噪声
- **DDIM Inversion 的确定性**是提取的核心依赖：只要图像未被几何变换，Inversion 能精确还原 w

**容量与参数：**
- `hw_factor=8`：marklength = 4×64×64 / (1×8×8) = **256 bits**，每 bit 64 次投票
- 无纠错编码，比特错误直接影响准确率

### 1.2 FDW（本方案）

FDW 在 GS 的基础上做了四项改进，目标是在保持即插即用的前提下**提升容量、增强鲁棒性、并解决几何攻击难题**。

**嵌入流程：**

```
水印比特 b ∈ {0,1}^512
  → 2× 重复码：[b | b] → 1024 bits（纠错层）
  → 空间冗余扩展（repeat 4×4 次）→ sd ∈ {0,1}^{4×64×64}
  → 流密码加密（ChaCha20）→ m ∈ {0,1}^{16384}
  → 截断高斯采样 → 初始噪声 w ∈ R^{4×64×64}
  → 频域中频增强（FD-Init）：
      W_fft = FFT2(w)
      T_fft = FFT2(m * 2 - 1)          # {0,1} → {-1,+1}
      W_fft[mid_freq] += λ · T_fft[mid_freq]   # λ=0.08
      w* = IFFT2(W_fft)，重归一化保持方差
  → 扩散生成（以 w* 为初始噪声）
      在 t*=0.3T 单步注入 X 形傅里叶模板（ShallowDiffuse 风格）
  → 带水印图像
```

**提取流程（含几何盲校正）：**

```
带水印图像（可能经过攻击）
  → [探针] VAE Encode + DDIM Inversion → z_T_probe
  → [缩放校正] 检测黑边对称性 → 裁剪内容 + resize 恢复 512×512
  → [旋转校正] z_T_probe 的 FFT 环形交叉相关 → 检测旋转角（精度 ≤0.5°）
                → 像素空间逆旋转
  → [正式提取] VAE Encode + DDIM Inversion → z_T
  → 符号检测：m̂ = (z_T > 0).int()
  → 流密码解密 → sd̂
  → 空间冗余投票 → b̂_full ∈ {0,1}^{1024}
  → 2× 重复码解码：b̂ = (copy1 OR copy2)
  → 与原始 b 比较，计算 Bit Accuracy
```

**四项核心改进：**

| 改进点 | 具体做法 | 作用 |
|--------|---------|------|
| **容量扩展** | hw_factor 从 8 降到 4，marklength 从 256→1024 | 有效载荷从 256→512 bits（2×） |
| **频域初始噪声增强（FD-Init）** | 在 w 的 FFT 中频段叠加水印模板信号（λ=0.08） | 中频段对模糊/压缩有天然不变性，提升降质攻击鲁棒性 |
| **ShallowDiffuse 单步模板注入** | 在 t*=0.3T 注入 X 形傅里叶模板（16个稀疏点） | 利用 Jacobian 低秩零空间，图像质量无损，模板用于几何定位 |
| **多级几何盲校正** | 黑边检测校正缩放 + FFT 环形交叉相关校正旋转 | 旋转/缩放攻击下水印准确率从 0.5→1.0 |
| **2× 重复码** | 512 bits payload 存两份，解码时 OR 融合 | 在 marklength=1024 空间内提供纠错 |

**容量与参数：**
- `hw_factor=4`：marklength = 4×64×64 / (1×4×4) = **1024 bits**，每 bit 16 次投票
- 2× 重复码后有效载荷 = **512 bits**，每 payload bit 实际有 32 次空间投票

## 三、核心结论

### 已验证的优势

**1. 容量翻倍，鲁棒性全面超越**

FDW 有效载荷 512 bits（GS 的 2 倍），同时在所有非组合攻击类别上优于 GS：

- JPEG Q=25：FDW 1.000 vs GS 0.909（**+9.1%**）
- 高斯模糊 r=4：FDW 0.985 vs GS 0.874（**+11.1%**）
- 裁剪 60%：FDW 0.915 vs GS 0.834（**+8.1%**）
- 缩放重采样 25%：FDW 1.000 vs GS 0.948（**+5.2%**）

**2. 几何攻击从完全失效到近完美**

这是 FDW 最重要的突破。通过 ShallowDiffuse 单步模板注入 + 多级几何盲校正，旋转和缩放攻击下的水印准确率从 GS 的约 0.5（随机水平）提升至约 1.0：

- 旋转 15°：0.505 → **0.998**
- 旋转 45°：0.486 → **1.000**
- 缩放 75%：0.519 → **1.000**

**3. 图像质量无损**

CLIP Score ≈ 0.33，与无水印生成持平。得益于 ShallowDiffuse 低秩理论——在 t*=0.3T 注入的信号处于 Jacobian 零空间，不改变生成过程。

**4. 即插即用，无需训练**

FDW 在 GS 基础上零额外训练成本，不修改模型权重。

### 当前主要短板

**Stirmark 组合攻击**（随机旋转 ±15° + 缩放 0.8-1.2× + 平移 ±20px）Acc≈0.5，因其随机平移破坏黑边检测，叠加多种变换增加校正难度。

## 四、关键技术决策记录

### 4.1 模板注入策略演进

| 阶段 | 策略 | 效果 |
|------|------|------|
| v1：最后1-2步注入 | gamma=20, t=[0.95, 1.0] | 信号太脆，不可靠 |
| v2：全50步注入 | gamma=0.5, t=[0.0, 1.0] | 信号强、可工作，但可能影响质量 |
| **v3（当前）：ShallowDiffuse 单步注入** | gamma=8, t*=0.3T | 信号充足（ratio≈8.7），图像质量无损，CLIP≈0.33 |

v3 的理论依据：在 t*=0.3T 处，PMP 的 Jacobian 是低秩的，注入信号大部分落在零空间，不改变预测 x_0。

### 4.2 旋转检测演进

| 阶段 | 方法 | 精度 |
|------|------|------|
| v1：稀疏点采样 | 整数 FFT bin 采样 | 1-3° 误差 |
| **v2（当前）：环形交叉相关** | 多半径双线性插值 → 1D 角度剖面 → 环形互相关 | **≤0.5°** |

### 4.3 几何校正空间选择

| 空间 | 结果 | 原因 |
|------|------|------|
| 潜空间（z_T）旋转 | 失败 | VAE 非旋转等变，信息损失 |
| **像素空间旋转** | **成功** | 校正后再 VAE 编码 + DDIM Inversion |

### 4.4 缩放 vs 裁剪区分

缩放攻击产生居中对称黑边，裁剪攻击产生随机位置黑边。通过检测黑边对称性（上下/左右边框差异 < 35%）来区分两者，避免缩放校正误触发裁剪攻击。
